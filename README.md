# Tuan Trinh – Maritime Strategy & Global Logistics Excellence

> Landing Page with integrated AI Chatbot (Trinh Bot AI)

## 🚀 Live Demo

Open `index.html` directly in any modern browser — no build step required.

## 📁 Project Structure

```
TuanTrinh2-landing-page/
├── index.html          # Main landing page + embedded chatbot
├── chatbot_data.txt    # Knowledge base for Trinh Bot AI
└── README.md
```

## 🤖 AI Chatbot Features

| Feature | Detail |
|---------|--------|
| **Provider** | OpenRouter API |
| **Model** | `z-ai/glm-4.5-air:free` |
| **UI Style** | Glassmorphism floating widget |
| **Markdown** | Rendered via `marked.js` |
| **Memory** | Multi-turn conversation history |
| **Animations** | Typing indicator, fade-in messages, refresh spin |
| **Responsive** | Mobile-friendly |

## 🛠️ How to Run

```bash
# Option 1: Open directly
open index.html

# Option 2: Local server (recommended for API calls)
python -m http.server 8080
# Then visit: http://localhost:8080

# Option 3: VS Code Live Server extension
```

## Tech Stack

- **HTML/CSS/JS** — No framework, no build step
- **Tailwind CSS** — via CDN
- **Google Fonts** — Noto Serif + Inter
- **marked.js** — Markdown rendering for chatbot responses
- **OpenRouter API** — AI backend
